# Cloud-Connected Security Node (IoT Telemetry)

## Description

This project is developed using an ESP32 microcontroller and an HC-SR04 ultrasonic sensor. The ESP32 measures the distance of nearby objects and transmits the sensor data to Adafruit IO using the MQTT protocol over Wi-Fi. The live distance values can be monitored remotely through the Adafruit IO dashboard, demonstrating real-time IoT telemetry and cloud connectivity.

## Components Used

- ESP32 Development Board
- HC-SR04 Ultrasonic Sensor
- Wi-Fi Network
- Adafruit IO
- MQTT (PubSubClient Library)

## How to Run

1. Open the project in Wokwi.
2. Install the required library (PubSubClient).
3. Enter your Wi-Fi SSID and password.
4. Enter your Adafruit IO Username and AIO Key.
5. Start the simulation.
6. Open the Adafruit IO dashboard to view the live distance data.
